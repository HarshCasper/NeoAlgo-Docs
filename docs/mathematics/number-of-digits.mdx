# Introduction

In this article we will see how to calculate the number of digits in a number.

# Algorithm

* We will make a cnt variable and initilize it to 0

* We will keep on dividing the number by 10 and increase the cnt variable until the number becomes zero

# Explanation 

### Example 1

```
 4568          cnt=0
 
 4568/10=456   cnt=1
 
 456/10=45     cnt=2
 
 45/10=4       cnt=3
 
 4/10=0        cnt=4
```
### Example 2

```
 789          cnt=0
 
 789/10=78    cnt=1
 
 78/10=7      cnt=2
 
 7/10=0       cnt=3
```

#include <bits/stdc++.h>
using namespace std;

int sum_of_digits(int num)
{
    int cnt = 0;
    while (num != 0)
    {
        cnt++;
        // Removing the least significant digit
        num = num / 10;
    }
    return cnt;
}

int main()
{
    int num, cnt;
    cout << "\nEnter the number: ";
    cin >> num;

    // If the number is negative, convert it into positive
    if (num < 0)
    {
        num = num * -1;
    }

    // Call the function
    cnt = sum_of_digits(num);
    // Print the obtained count
    cout << "The number of digits in the given number is: " << cnt;
    return 0;
}
