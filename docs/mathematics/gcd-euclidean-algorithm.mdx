## Introduction

The GCD of two numbers is the largest integer that divides each of the integers such that their remainder is zero. The Euclidean algorithm is an efficient way for finding the GCD of `2` numbers. It is named after the Greek mathematician Euclid

## Explanation

Consider two numbers `a = 333` and `b = 84`

`333 = 3*84 + 81`   `0 < 81 < 84`

`84 = 1*81 + 3`     `0 < 3 < 81`

`81 = 27*3 + 0`

Since 3 is the last nonzero remainder, the euclidean algorithm tells us that GCD(a,b) = 3.

## Algorithm

- input `a` and `b`
- repeat steps until a mob b is not 0 
    - `r = a mod b`
    - `a = b`
    - `b = r`
- return `b`

#include<bits/stdc++.h>
using namespace std;

int gcd(int a,int b)
{ // base case
  if (a==0)
     return b;
  // recursive case
  else
     return gcd(b%a,a);
}

// initializing a and b.
// Here a is always the smaller number and
// b is always the bigger number between the two.

int main()
{
    cout<<"Enter a and b for finding GCD"<<endl;
    int a,b;
    cin>>a>>b;
    cout<<"GCD( "<<a<<" , "<<b<<" ) = "<< gcd(a,b)<<endl; 
    return 0;
}

def gcd_of_nums(x,y):
    # choose the smaller number
    if x > y:
        t=x
        x=y
        y=t

    if(x==0):
        return y
    else:
        return gcd_of_nums(x,y%x)

def main():
    print("Enter a and b for finding GCD")
    a,b = map(int,input().split(' '))
    print("GCD( "+str(a)+" , "+str(b)+" ) = "+str(gcd_of_nums(a,b)))
    
if __name__ == "__main__":
    main()
