## Introduction

The reverse of a number is a number that is obtained when a number is traversed from right to left.

## Explanation 

We will make a reverse variable and initialize it to `0`. Then we will keep on extracting the least significant digit from the number by doing the modulus by `10` and adding it to the reverse variable after multiplying it by `10` till the number becomes `0`. 

Let's have one example:
```
num=4568          
rev=0

rev = rev*10 + num%10 = 8
num = num/10 = 456 

rev = rev*10 + num%10 = 86
num = num/10 = 45

rev = rev*10 + num%10 = 865
num = num/10 = 4 

rev = rev*10 + num%10 = 8654
num = num/10 = 0
```

## Algorithm

* We will make a rev variable and initialize it by `0`.

* We will extract the least significant digit by doing modulus by `10` and remove the last digit by dividing by `10`.

* Now we will multiply the rev variable by `10` and add extracted element to it.

* Apply the previous two steps until the number becomes `0`.

```cpp
#include <bits/stdc++.h>

using namespace std;

int reverse(int num) {
  int rev = 0;
  do {
    rev = rev * 10 + num % 10;
    num = num / 10;
  } while (num > 0);

  return rev;
}

int main() {
  int num;
  cout << "Enter the number: ";
  cin >> num;

  cout << "The reverse of the given number is: " << reverse(num);
  return 0;
}
```

def reverse(n):
    rev = 0
    while True:
        if n == 0:
            break
        rev = rev * 10 + n % 10
        n = n // 10
    return rev


n = int(input('Enter the number :'))

print 'The reverse of the given number is: {} '.format(reverse(n))
