## Introduction

The hamming distance is the value difference between the two binary representations of the 32-bit integer.

## Explanation

If we are given two numbers then the hamming distance is the number of positions at which the corresponding symbols differ.

Let's have one example: 
```
Input :
num1 = 7(0111)
num2 = 8(1000)

Output : 4
```

## Algorithm

- Take the given two digits as input from the user.
- Compute the binary equivalent of the two decimal numbers.
- Compute the number of positions at which the corresponding binary equivalent numbers differ.

#include<bits/stdc++.h> 
using namespace std;

int solve1(int x, int y) {

    int ans = x^y;
    return __builtin_popcount(ans); 
}

int solve2(int x, int y) {

    int count = 0;
    for (int i = 0; i < 32; i++)
        if ( (x & (1 << i)) != (y & (1 << i)) )
            count++;
    return count;
}

def hamming_distance(num1, num2):

    #To store value of the differnce
    bitdiff = 0
    
    for i in range(31, -1, -1):
        
        #right shift for both the numbers for i AND 1 times
        binary_num1 = num1>>i&1
        binary_num2 = num2>>i&1

        #checking if binary of both are not equal
        if binary_num1 != binary_num2:
            bitdiff += 1
        #i it is equal bitdiff will be same and loop will continue

    return bitdiff

def main():

    #User input for a both numbers
    num1 = int(input("Enter 1st number: "))
    num2 = int(input("Enter 2nd number: "))

    #function call for getting hamming distance for the given two numbers.
    print("Hamming distance between these two numbers is: ", hamming_distance(num1, num2))

if __name__ == "__main__":
    main()
