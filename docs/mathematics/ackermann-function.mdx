## Introduction

Ackermann function is one of the simplest examples of a total computable function,
that is not primitive recursive.

## Explanation

It is basically a function with two non-negative integers as arguments.

Ackermann function can be declared as:
```
A(0, j)=j+1 for j â‰¥ 0
A(i, 0)=A(i-1, 1) for i > 0
A(i, j)=A(i-1, A(i, j-1)) for i, j > 0
```

Let's have few examples: 
```
For i=1,j=2
Output=4

For i=3,j=3
Output=61
```

## Algorithm

```
Ackermann_Function(int i, int j):
  - Checking whether `i` is `0`,if yes 
    then applying the associated condition.
  - Checking whether `i` is greater than `0` and 
    `j` is `0` , if yes then calling the function
    with `m-1` and `1`.
  - Checking whether `i` and `j` are greater than `0`, if yes
    then applying the associated condition.
```

#include<bits/stdc++.h>
using namespace std;
int Ackermann_Function(int m, int n) {
	if (m == 0)
		return n + 1;

	else if (m > 0 && n == 0)
		return Ackermann_Function(m - 1, 1);

	else if (m > 0 && n > 0)
		return Ackermann_Function(m - 1, Ackermann_Function(m, n - 1));
}

int main() {
	int m, n;
	cout << "Input first Number\n";
	cin >> m;
	cout << "Input second Number\n";
	cin >> n;
	cout << "Result: " << Ackermann_Function(m, n);
	return 0;
}
