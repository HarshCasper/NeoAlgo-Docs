## Introduction

The Ackermann function is one of the classic example of a recursive function, that is not a primitive recursive function. It grows very quickly in value, faster than an exponential function.

## Explanation

It is basically a function with two non-negative integers as arguments.
Assuming `i` and `j` as the non-negative integers.
Ackermann function can be declared as:
```
A(0, j)=j+1 for j â‰¥ 0
A(i, 0)=A(i-1, 1) for i > 0
A(i, j)=A(i-1, A(i, j-1)) for i, j > 0
```
Let's have few examples: 
```
For i=1,j=2
Output=4

For i=3,j=3
Output=61
```

## Algorithm

```
- Taking two non-negative integers from user.
- Calling the function with the input values.
Ackermann_Function(int i, int j):
  - Checking the values of `i` and `j`, 
    then applying the associated conditions.
```

#include<bits/stdc++.h>
using namespace std;
int Ackermann_Function(int m, int n) {
	if (m == 0)
		return n + 1;

	else if (m > 0 && n == 0)
		return Ackermann_Function(m - 1, 1);

	else if (m > 0 && n > 0)
		return Ackermann_Function(m - 1, Ackermann_Function(m, n - 1));
}

int main() {
	int m, n;
	cout << "Input first Number\n";
	cin >> m;
	cout << "Input second Number\n";
	cin >> n;
	cout << "Result: " << Ackermann_Function(m, n);
	return 0;
}

## Complexity Analysis

- Time  complexity - `O(mA(m, n))` to calculate `A(m,n)`
- Space complexity - `O(m)` to calculate `A(m, n)`
  Here, `m` and `n` are the two integers that we take from user
