## Introduction

The Ackermann function is multi-variable function from the natural numbers to the natural numbers with very fast rate of growth. It is one of the classic example of a recursive function, that is not a primitive recursive function. All primitive recursive functions are total and computable, but the Ackermann function illustrates that not all total computable functions are primitive recursive.

## Explanation

It is basically a function with two non-negative integers as arguments.
Assuming `i` and `j` as the non-negative integers.
Ackermann function can be declared as:
```
A(0, j)=j+1 for j â‰¥ 0
A(i, 0)=A(i-1, 1) for i > 0
A(i, j)=A(i-1, A(i, j-1)) for i, j > 0
```
Let's have an example to see how the Ackermann function grows so quickly: 
```
A(1,2)  = A(0, A(1, 1)) 	// applying third condition of Ackermann function
	= A(0, A(0, A(1, 0))) 	// applying third condition of Ackermann function
	= A(0, A(0, A(0, 1))) 	// applying second condition of Ackermann function
	= A(0, A(0, 2)) 	// applying first condition of Ackermann function: A(0, 1) = 2
	= A(0, 3) 		// applying first condition of Ackermann function: A(0, 2) = 3
	= 4.			// applying first condition of Ackermann function: A(0, 3) = 4
```

## Algorithm

```
Ackermann(i, j):
  - Checking the values of `i` and `j` 
  - Applying the associated conditions
  - Recursively calling the function until the first condition is applied
  - Obtaining the final result
  - Returning the resultant value
```

#include<bits/stdc++.h>
using namespace std;
int Ackermann_Function(int m, int n) {
	if (m == 0)
		return n + 1;

	else if (m > 0 && n == 0)
		return Ackermann_Function(m - 1, 1);

	else if (m > 0 && n > 0)
		return Ackermann_Function(m - 1, Ackermann_Function(m, n - 1));
}

int main() {
	int m, n;
	cout << "Input first Number\n";
	cin >> m;
	cout << "Input second Number\n";
	cin >> n;
	cout << "Result: " << Ackermann_Function(m, n);
	return 0;
}

## Complexity Analysis

- Time  complexity - `O(mA(m, n))` to calculate `A(m,n)`
- Space complexity - `O(m)` to calculate `A(m, n)`
  Here, `m` and `n` are the two integers that we take from user
