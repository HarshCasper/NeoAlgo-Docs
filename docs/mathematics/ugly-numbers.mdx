## Introduction

Ugly Numbers are numbers that have only `2`, `3`, and `5` as their prime factors. Prime factors are the prime numbers that perfectly divide the given number. Prime Number refers to the number which is divisible by `1` and the number itself. `2` is the smallest prime number. `1` is neither prime nor composite.  

## Explanation 

To check whether the number is ugly or not we just need to keep on dividing it by `2`, `3`, and `5`. If it reduces to `1`, then the number is ugly else the number is not ugly. Prime factorization of every ugly number looks like: 2<sup>p</sup> x 3<sup>q</sup> x 5<sup>r</sup>, where `p >= 0`, `q >= 0`, and `r >= 0`. 

`1` is included in the list of ugly numbers by convention. First `10` ugly numbers are `1`, `2`, `3`, `4`, `5`, `6`, `8`, `9`, `10`, `12`.

Let's have one example: 
```
num = 45

It is not divisible by 2, so we will not divide it by 2

It is divisible by 3, so we will divide it by 3 till it becomes not divisible
num = 45/3 = 15
num = 15/3 = 5

It is divisible by 5, so we will divide it by 5 till it becomes not divisible
num = 5/5 = 1

It reduces to 1, so it is a Ugly Number.
```

## Algorithm

- Divide the number by `2` until it becomes not divisible by `2`.

- Divide the number by `3` until it becomes not divisible by `3`.

- Divide the number by `5` until it becomes not divisible by `5`.

- Return `true` if the reduced number is `1`, else return `false`.

using namespace std;
// Function to determine whether the number is ugly or not

int isUgly(int n) 
{
    while (true) 
    {
        if (n % 2 == 0) n = n / 2;
        if (n % 3 == 0) n = n / 3;
        if (n % 5 == 0) n = n / 5;
        if (n % 2 != 0 && n % 3 != 0 && n % 5 != 0) 
        {
            break;
        }
    }
    if (n == 1) return true;
    else return false;
}

int main() 
{
    cout << "Enter the Number: ";
    int n;
    cin >> n;

    if (isUgly(n)) 
    {
        cout << "Ugly Number" << endl;
    } 
    else 
    {
        cout << "Not a Ugly Number" << endl;
    }
    return 0;
}
