## Introduction

In this article, we will see how to find the minimum element in a given set of data. Minimum element means the smallest value in a given array.

## Explanation 

In this, we have to find the minimum element or we can say smallest value from the given set of data. For that first we assume that first element of our array is smallest and store into a res variable. Keep updating the res variable when we see smaller element than the previous one.

## Algorithm

For finding Maximum element in array, we follow the following steps:

* Initialise a variable res with first element in array.

* Iterate on whole array.

* Keep updating res variable when we see current element is smaller than res.

* Return the res which contain minimum element.

```cpp
#include<bits/stdc++.h>
using namespace std;

//function minimum_element 
int maximum_element(int a[],int n)
{
  int res=a[0];
  
  //for finding maximum element in array
  for(int i=1;i<n;i++)
  {
    if(a[i]<res)
    {
      res=a[i];
    }
  }
  
  //after finding res return
  return res;
}
```

```python   
# function minimum_element
def minimum_element(a, n):
 
    # initialize count with 0
    count = a[0]
    i = 1
    while n != 1:
        if a[i] < count:
            count = a[i]
            i += 1
            n -= 1
 
    return count
 
 
# Taking size as input
n = int(input("Enter the size of array: "))
 
#  Taking elements of array as input
print("Enter the elements of array: ")
data = []
```
