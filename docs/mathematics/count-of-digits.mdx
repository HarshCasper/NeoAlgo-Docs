## Introduction

Count of digits refers to the sum of the frequency of all distinct digits in a number.

## Explanation 

We can remove the least significant digit of a number by dividing it by `10`. We have to continuously reduce the digits of a number and increase the count till the number becomes `0`.

Let's have one example: 
```
num = 4568         
cnt = 0 
 
num = 4568/10=456   
cnt = 1
 
num = 456/10=45     
cnt = 2
 
num = 45/10=4       
cnt = 3
 
num = 4/10=0        
cnt = 4
```

## Algorithm

* We will make a `cnt` variable and initialize it to `0`.

* We will keep on dividing the number by `10` and increase the `cnt` variable until the number becomes `0`.

#include <bits/stdc++.h>
using namespace std;

int count_of_digits(int num)
{
    if(num==0)return 1;  
    int cnt = 0;
    while (num != 0)
    {
        cnt++;
        // Removing the least significant digit
        num = num / 10;
    }
    return cnt;
}

int main()
{
    int num, cnt;
    cout << "Enter the number: ";
    cin >> num;

    // If the number is negative, convert it into positive
    if (num < 0)
    {
        num = num * -1;
    }

    // Call the function
    cnt = count_of_digits(num);
    // Print the obtained count
    cout << "The count of digits in the given number is: " << cnt;
    return 0;
}

def count_of_number(n):
    if(n==0):
      return 1
    #Initializing cnt variable to zero
    cnt = 0
    while True: 
        if(n==0):
          break 
        #Increasing the cnt
        cnt+=1 
        #Removing the least significant digit
        n = n//10
    return cnt
