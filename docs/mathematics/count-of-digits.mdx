## Introduction

Count of digits refers to the sum of the frequency of all distinct digits in a number. The number having a higher count of digits will be greater than the number having a lower count of digits. In `C++` we can have a number having at max 20 count of digits. In `Python` there is no limits on the count of digits in a number.

## Explanation 

We need to keep on removing the least significant digit of a number and increase the count till the number becomes zero. All numbers in the range `[0,9]` have the count of digits as `1`. All numbers in the range `[10,99]` have the count of digits as `2`. All numbers in the range `[100,999]` have the count of digits as `3`.

Let's have one example: 
```
num = 4568         
cnt = 0 
 
Iteration 1:
num = 4568/10 = 456   
cnt = cnt + 1 = 1
 
Iteration 2:
num =  456/10 = 45     
cnt = cnt + 1 = 2
 
Iteration 3:
num =  45/10  = 4       
cnt = cnt + 1 = 3
 
Iteration 4:
num =  4/10   = 0        
cnt = cnt + 1 = 4 
```

## Algorithm

* Make a `cnt` variable and initialize it to `0`.

* Keep on dividing the number by `10` and increase the `cnt` variable by `1` until the number becomes `0`.

#include <bits/stdc++.h>
using namespace std;

//Function to count digits in the number

int count_of_digits(int num)
{
    if(num==0)return 1;  
    int cnt = 0;
    while (num != 0)
    {
        //Incrementing cnt by 1

        cnt++;

        // Removing the least significant digit

        num = num / 10;
    }
    return cnt;
}

int main()
{
    int num, cnt;

    //Taking Input 

    cout << "Enter the number: ";
    cin >> num;

    // If the number is negative, convert it into positive

    if (num < 0)
    {
        num = num * -1;
    }

    // Call the function

    cnt = count_of_digits(num);

    // Print the obtained count

    cout << "The count of digits in the given number is: " << cnt;
    return 0;
}
