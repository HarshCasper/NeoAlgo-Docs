## Introduction

In this article we will see how to calculate the count of digits in a number.

## Explanation 

We will make a cnt variable and initilize it to `0`. We will keep on dividing the number by `10` and keep on increasing the cnt variable until the number becomes `0`.

Let's have one example: 
```
 4568          cnt=0
 
 4568/10=456   cnt=1
 
 456/10=45     cnt=2
 
 45/10=4       cnt=3
 
 4/10=0        cnt=4
```

## Algorithm

* We will make a cnt variable and initilize it to `0`.

* We will keep on dividing the number by `10` and increase the cnt variable until the number becomes `0`.

#include <bits/stdc++.h>
using namespace std;

int count_of_digits(int num)
{
    int cnt = 0;
    while (num != 0)
    {
        cnt++;
        // Removing the least significant digit
        num = num / 10;
    }
    return cnt;
}

int main()
{
    int num, cnt;
    cout << "Enter the number: ";
    cin >> num;

    // If the number is negative, convert it into positive
    if (num < 0)
    {
        num = num * -1;
    }

    // Call the function
    cnt = count_of_digits(num);
    // Print the obtained count
    cout << "The count of digits in the given number is: " << cnt;
    return 0;
}
