## Introduction

In this article, we will see how to find the maximum element in a given set of data. Maximum element means the largest value in a given array.

## Explanation 

In this, we have to find the maximum element or we can say largest value from the given set of data. For that first we assume that first element of our array is largest and store into a res variable. Keep updating the res variable when we see greater element than the previous one.

```
arr[] = {2, 4, 3, 8, 17, 14, 10, 6}

Return the res which is maximum element in array.

```
## Algorithm

For finding Maximum element in array, we follow the following algorithm:

* Initialise a variable res with first element in array.

* Iterate on whole array.

* Keep updating res variable when we see current element is greater than res.

* Return the res which contain maximum element.

```cpp
#include<bits/stdc++.h>
using namespace std;

//function maximum_element 
int maximum_element(int a[],int n)
{
  int res=a[0];
  
  //for finding maximum element in array
  for(int i=1;i<n;i++)
  {
    if(a[i]>res)
    {
      res=a[i];
    }
  }
  
  //after finding res return
  return res;
}
```

```python   
# function maximum_element
def maximum_element(a, n):
 
    # initialize count with 0
    count = a[0]
    i = 1
    while n != 1:
        if a[i] > count:
            count = a[i]
            i += 1
            n -= 1
 
    return count
 
 
# Taking size as input
n = int(input("Enter the size of array: "))
 
#  Taking elements of array as input
print("Enter the elements of array: ")
data = []
for i in range(n):
    data.append(int(input()))
```
