## Introduction

Factorial of a number `n` is defined as the product of all positive integers less than or equal to `n`. Factorial function is defined for all non-negative integers. The value of `0!` is `1`.

## Explanation

For a non-negative integer say `n`, factorial of `n` is defined as follows:

```
Factorial(n) = n * (n - 1) * (n - 2) * . . . . . 3  * 2 * 1
```

It is also defined as:

```
Factorial(n) = n * Factorial(n - 1)
```

Let us have a few examples:

```
Factorial(5) = 5 * 4 * 3 * 2 * 1 = 120
Factorial(6) = 6 * 5 * 4 * 3 * 2 * 1 = 720 = 6 * Factorial(5)
```

## Algorithm

### Recursive

For a given number `n`,

 - If `n` == `1`, return `1`.
 - Else recursively return `n` * `factorial(n - 1)`

```
factorial(n):
    if(n == 1)
        return 1
    else
        return n * factorial(n - 1)
```

### Iterative

For a given number `n`,

 - Declare a variable factorial and initialize it to `1`.
 - For `num` starting from `1` to `n`.
 - Compute `factorial` = `factorial` * `num`
 - Add the end of the for loop, return the `factorial`.

```
factorial = 1
for i = 1 to n
  factorial = factorial * i
return factorial
```

//The length of vector should be >= length of the resultant factorial
#include<bits/stdc++.h>
using namespace std;

/*The function takes a number and multiplies the number in the school's old fashioned
way. Herein we multiply the complete digit by one digit and then add the results to get
the final resultant. The carry is carried forwarded and in the end the final result gets
stored in the array/vector. */

void factorial(int val)
{
    int carry = 0;       
    vector <int> arr(10000, 0);   
    arr[0] = 1;                   
    int k = 0;                         

    for(int i = 1; i <= val; i++) {
        for(int j = 0;j <= k; j++) {      
            arr[j] = arr[j] * i + carry;
            carry = arr[j] / 10;
            arr[j] = arr[j] % 10;
        }
        while(carry) {        
            k++;
            arr[k] = carry % 10;
            carry /= 10;
        }   
    }
    for(int i = k; i >= 0; i--) {  
        cout<<arr[i];
    }
    cout << "\n";
}
int main() {
    int num;
    cout << "Enter the number: ";
    cin>>num;
    cout << "\nThe factorial of " << num << " is: ";
    factorial(num);
    return 0;
}

# Iterative implementation for factorial
def fact(x):
    result = 1
    for i in range(1,x):
        result = result * (i+1)
    return result

# Calculate factorial using recursion
def fact_rec(x):
    if x == 0 or x == 1:
        return 1
    else:
        return x * fact_rec(x-1)


if __name__ == "__main__":
    n = int(input("Enter the number: "))
    print("The factorial of {} is: {} ".format(n, fact(n)))
