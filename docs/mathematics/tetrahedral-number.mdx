## Introduction

A Tetrahedral number is a figurate number. A Figurate number is a number that can be represented by a regular geometrical arrangement of equally spaced points. The Tetrahedral number represents a pyramid with a triangular base and three sides. We are given a number `n` and we need to print the `nth` tetrahedral number.

## Explanation 

The `nth` Tetrahedral number is the sum of first `n` triangular numbers. For calculating the `nth` tetrahedral number just apply the formula given below. First few tetrahedral numbers are `1`, `4`, `10`, `20`, `35`, `56`, `84`, `120`, `165`, `220`.

<img src="https://i.imgur.com/lSdNIiO.jpg" width="272" height="189" /><br/><br/>

Formula: 
```
nth Tetrahedral Number: (n * (n + 1) * (n + 2)) / 6
```

Let's have few examples: 
```
n = 3
3rd tetrahedral number = 10

n = 5
5th tetrahedral number = 35

n = 7
7th tetrahedral number = 84
```

## Algorithm

- Return `(n * (n + 1) * (n + 2)) / 6`.

#include<bits/stdc++.h>
using namespace std;
// Function to return nth tetrahedral number

int tetrahedral_number(int n) 
{
    int res = n * (n+1) * (n+2);
    res = res / 6;
    return res;
}

int main() 
{
    int n;
    cout << "Enter the Number: ";
    cin >> n;

    cout<<"Tetrahedral Number is: "<<tetrahedral_number(n)<<endl;
