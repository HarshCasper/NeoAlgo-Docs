## Introduction

Greatest Common Divisor(GCD) is highest number which is factor of both the numbers. It is also called as Highest Common Factor(HCF). Two numbers are coprime if there GCD is 1. GCD of two even numbers is even and odd numbers is odd.

## Explanation 

For a non-negative integer say `n`, Double factorial of `n` is defined as follows:

If `n` is even:
```
DoubleFactorial(n) = n * (n - 2) * . . . . . 4  * 2
```

If `n` is odd:
```
DoubleFactorial(n) = n * (n - 2) * . . . . . 3  * 1
```

Let's have few examples: 

```
num1 = 6
num2 = 9
mini = 6, it contains minimum of num1 and num2

Iteration 1:
mini = 6
Both numbers are not divisible by mini, so reduce mini by 1
mini = mini-1

Iteration 2:
mini = 5
Both numbers are not divisible by mini, so reduce mini by 1
mini = mini-1

Iteration 3:
mini = 4
Both numbers are not divisible by mini, so reduce mini by 1
mini = mini-1

Iteration 4:
mini = 3
Both the numbers are divisible by mini, so return mini
```

## Algorithm

- Declare a variable `mini` and initialize it to minimum of two numbers.

- Decrease `mini` until both the numbers become divisible by it.

- Return `mini`.

#include<bits/stdc++.h>
using namespace std;
// Function to determine the gcd of two numbers

int gcd(int a, int b) 
{
    // Assigning minimum element to mini

    int mini;
    if (a <= b) mini = a;
    else mini = b;

    while (mini > 1) 
    {
        if (a % mini == 0 && b % mini == 0) 
        {
            break;
        }
        mini--;
    }
    return mini;
}
