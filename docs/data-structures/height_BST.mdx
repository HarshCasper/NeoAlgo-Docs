## Introduction

In this article, we will see how to find the height of binary search tree(BST). We are given an Binary Search Tree(BST) and we have to find the height. In this type of tree(BST), left child is always smaller than its parent and right child is always greater than its parent.

## Explanation 

The height of a binary search tree is the maximum distance from the root node to the leaf node or equivalently, the depth of its deepest node. Height of the empty tree is zero.

## Algorithm

For calculating Height of BST, we follow the following steps:

* First, we check whether tree is empty or not.

* If tree is empty, then simply return height zero. 

* Otherwise calculate height of BST that is maximum distance from root node to leaf node.

* Recursively calculate maximum of height of left child and right child and simply add 1(for root node).

//creating structure for node
struct Node {
    int data;
    struct Node *left;
    struct Node *right;
};

// Insert Function
Node* Insert(Node *root, int data) {
    //if root have no value
    //then create a node and insert values
    if (root == NULL) {
        root = new Node();
        root->data = data;
        root->left = root->right = NULL;
    }
    //check if data is less than root value
    //then insert into left subtree
    else if (data <= root->data)
        root->left = Insert(root->left, data);
    //check if data is less than root value
    //then insert into right subtree
    else
        root->right = Insert(root->right, data);
    return root;
}

//heightofbst Function
int heightofbst(Node *root) {
    // if tree is empty then return
    if (root == NULL) 
    {
      return 0;
    }
    else
    {
      return max(heightofbst(root->left),heightofbst(root->right))+1;
    }
    
}

class Node:

    def __init__(self, key):
        self.left = None
        self.right = None
        self.value = key


# A function to insert a new node with the given key value

def insert(root, node):
    if root is None:
        root = node
    else:
        if root.value < node.value:
            if root.right is None:
                root.right = node
            else:
                insert(root.right, node)
        else:
            if root.left is None:
                root.left = node
            else:
                insert(root.left, node)


# A function for calculating height of bst 

def heightofbst(root):
    if root is None:
        return 0 ;
    else :
      return max(heightofbst(root.left),heightofbst(root.right))+1;
