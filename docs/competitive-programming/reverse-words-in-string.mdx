## Introduction

Given a string, we need to reverse the whole string without reversing the individual words in it.

## Explanation

In this algorithm we have to reverse each word of a string. In other words, we have to reverse text without reversing individual word in a string.
Let us have a few examples:
```
Reverse of "i like this program very much" will be "much very program this like i"
Reverse of "pqr mno" will be "mno pqr"
```

## Algorithm

#### Using Stack

```
- declaring an empty stack and a string.
- adding each character of string,that is not a "space" to the declared string.
- pushing declared "string" and "space" character in the stack.
- using while loop, storing the top character in result string and popping out every single character.
```

#### Iterative

```
- creating a string array to store result.
- using for loop, extracting each word from the string and storing in the array.
- using for loop, printing the words of string in reverse order.
```

#include<bits/stdc++.h>
using namespace std;
int main() {
   
    stack<string> str_stack;
    string str, stack_string;

    cout << "Enter the string: " << endl;
    getline(cin,str); 

    for (int i = 0 ; i < str.size() ; i++) {
        if (str[i] != ' ') {
            stack_string += str[i];
        } else {
            str_stack.push(stack_string);
            str_stack.push(" ");
            stack_string = "";
        }
    }
    str_stack.push(stack_string);
    str = "";
    while (!str_stack.empty()) {
        str += str_stack.top();
        str_stack.pop();
    }
    cout<<"String after Reverse:\n";
    cout << str << endl;
    return (0);
}

## Complexity Analysis

- Time complexity - `O(n)` 
- Space complexity - `O(n)`
- Here in "stack" approach `n` denotes the length of the string, whereas in "iterative" approach `n` denotes the number of words in the string.
