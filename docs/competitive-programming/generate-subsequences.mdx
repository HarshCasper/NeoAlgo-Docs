## Introduction

Given a string containing n characters, find out all the subsequences of the string.

## Explanation 

We can do it by first taking an empty string. Initially, there will two seperate cases, 
Firstly, including the current character in the output string and then recursively calling the function.
The second will be, excluding the current encountered character and then recursively calling the function.

Let's have few examples: 
```
Input 1:
Enter string:
abc
Output 1:
The subsequences are: abc ab ac a bc b c
Input 2:
Enter string:
abcd
Output 2:
The subsequences are: abcd abc abd ab acd ac ad a bcd bc bd b cd c d
```

## Algorithm

- Initialize an empty array(out)
    - If the input string reaches its end
        - Add a NULL character at the end
	- Print the subsequence

    - Include the current character out[j]=in[i]
	- recursively call the function

    - Exclude the current character
	- recursively call the function

#include<bits/stdc++.h>
using namespace std;

void generate_subsequence(char *in,char *out,int i,int j)
{
	if(in[i]=='\0')
	{
		out[j]='\0';
		cout<<out<<" ";
		return;
	}
	out[j]=in[i];
	generate_subsequence(in,out,i+1,j+1);

	generate_subsequence(in,out,i+1,j);
}

int main()
{
	char input[100];
	char output[10];
	cout<<"Enter string: "<<endl;
	cin>>input;
	cout<<"The subsequences are: ";
	generate_subsequence(input,output,0,0);
	return 0;
}

# Generate all subsets of a string
def printSubsets(str,output):
    if len(str)==0:
        print(output)
        return
    printSubsets(str[1:],output)
    newOutput=output+str[0]
    printSubsets(str[1:],newOutput)
    return
