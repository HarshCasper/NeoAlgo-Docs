## Introduction

Given a string, we need to find its reverse.

## Explanation

Reversing a string implies that the string given by the user in a specific sequence will get entirely reversed,
when the reverse of a string algorithm is implemented.

Let us have a few examples:

```
Reverse of "hello" will be "olleh"
Reverse of "NeoAlgo" will be "oglAoeN"
```

## Algorithm

#### Using Stack

```
reverse(s):
  - creating an empty stack.
  - pushing each character of string in stack.
  - using for loop, storing the top character in result string 
    and popping out every single character.
```

#### Iterative

```
reverse(s):
  - creating a variable to store result.
  - using for loop, extracting each element from the end of the string 
    and storing in the variable.
  - printing the reversed string.
```

#include<bits/stdc++.h>
using namespace std;

//function to reverse the string
void reverse(string &s)
{
	int i;
	stack<int> str;

	for(char ch:s){
        str.push(ch);
    }
    for(i=0;i<s.size();i++){
        s[i]=str.top();
        str.pop();
    } 
}

int main()
{
	string s;
	cout<<"Enter the string : ";
	getline(cin,s);
	reverse(s);
   	cout<<"String after Reverse: ";
	cout<<s;
	return 0;
}

import java.io.*;

class Program {

        String str = sc.nextLine();

        // revoking the function reverse

        reverse(str);
        
    }

    // Method for reversing the string

    static void reverse(String s){

        // Reversing the string by extracting each element from the end of the string
        // and storing in a variable result

        String result = "";
        for (int i = s.length() - 1; i >= 0; i--)
            result += s.charAt(i);

        // Printing the reversed string
        System.out.println("String after Reverse: ");
        System.out.println(result);
       
    }

}
