# Introduction

Selection Sort is a sorting algorithm which finds out the smallest element in unsorted part in each iteration and put it to the last of sorted part.
It does less memory writes as compared to other sorting algorithms.

# Algorithm

In Selection Sort, we follow the following algorithm:

* First we will find the smallest element in the range [0,n-1] and swap it with the element at 0th index

* Then we will find the smallest element int the range [1,n-1] and swap it with the element at 1st index

* We keep on increasing the left part of the range until it reaches n-1 and do the above mentioned process

```cpp
#include<bits/stdc++.h>
using namespace std;

void selection_sort(int *array, int size) {
   int i, j, imin;
   for(i = 0; i<size-1; i++) {
      imin = i;   //get index of minimum data
      for(j = i+1; j<size; j++)
         if(array[j] < array[imin])
            imin = j;
         //placing in correct position
         swap(array[i], array[imin]);
   }
}
```

```python   
def selection_sort(nums):
    for i in range(len(nums)):
        lowest_value_index = i
        for j in range(i + 1, len(nums)):
            if nums[j] < nums[lowest_value_index]:
                lowest_value_index = j
        nums[i], nums[lowest_value_index] = nums[lowest_value_index], nums[i]
```

# Complexity Analysis
 
`n` is the number of elements in the array.

- Time Complexity:

  - **Best Case**: O (n)
  - **Average Case**: O (n<sup>2</sup>)
  - **Worst Case**: O (n<sup>2</sup>)

- Space Complexity: O (1)
